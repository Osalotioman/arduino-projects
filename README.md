## Gas Leak Detector
[Source_Code](https://github.com/Osalotioman/arduino_projects/blob/main/GasDetector/GasDetector.ino) <br>
**Overview**: A smart gas detection system that uses an MQ sensor to monitor gas levels.
- **Alerts**: Sends SMS ("EMERGENCY!!!!") and places calls via a GSM module when leaks are detected.
- **Alarm**: Activates a buzzer with a frequency proportional to the gas intensity.
